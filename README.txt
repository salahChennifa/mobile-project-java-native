		  calendarPart
                =============
calendarPart est conçu pour fournir une visualisation familière d'un jour calendaire. 
Le rendu est effectué par la classe `DayView`, qui prend une liste d'événements et les 
affiche à l'aide d'un algorithme de mise en page personnalisé.
